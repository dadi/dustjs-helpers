# DADI Dust.js helpers

> A collection of Dust.js helpers used in DADI+ projects.

## Development

1. Clone this repository

2. Make your changes to `dadi-dustjs-helpers.js`

3. Add your tests to `test/test.js`

4. Run test suite
   
   ```
   npm test
   ```

5. Build minified file

   ```
   npm run build
   ```
